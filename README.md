せきとめ
========

落ちてくるnil2を落とさないようにするゲーム。

操作方法
--------

### 一般

| 操作         | キー        |
|--------------|-------------|
| 上下左右移動 | 矢印キー    |
| 選択         | Enter or Z  |
| キャンセル   | Escape or X |

### ゲーム

| 操作       | キー                              |
|------------|-----------------------------------|
| 左右移動   | 矢印キー                          |
| リスタート | R                                 |
| 終了       | Escape or X(ゲームオーバー後のみ) |
